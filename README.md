Marketing Campaign Analysis - Business Insights Report

Task 04 - SkillCraft Technology Data Science Internship

Project Overview

This project performs Exploratory Data Analysis (EDA) on a marketing campaign dataset to uncover customer behavior, purchasing patterns, and campaign performance. The analysis includes data cleaning, visualization, and business recommendations.

Tools Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

Key Insights

- Analyzed customer income and age distribution.
- Explored education and marital status.
- Compared spending across product categories.
- Evaluated marketing campaign acceptance.
- Analyzed purchase channels.
- Generated a correlation heatmap for numerical features.

Files

- SCT_DS_4.ipynb

Outcome

The project provides actionable business insights and recommendations to improve future marketing campaigns.
